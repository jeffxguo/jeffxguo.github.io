# **App Name**: Dev Showcase

## Core Features:

- Project Gallery: Interactive gallery to display development projects with cards containing descriptions, technologies used, and external links.
- About Me Section: Dedicated 'About Me' section detailing personal and professional background.
- Contact Section: Contact section with a form for users to send direct messages.

## Style Guidelines:

- Color scheme: Dark background for a modern feel. In HSL, hue 210, saturation 20%, lightness 10% yields hex #1A1D24.
- Primary color: Professional blue (#5591F9) to be used for primary interactive elements. In HSL, hue 220, saturation 90%, lightness 66% yields hex #5591F9.
- Accent color: A lighter blue (#7DC4FF) as an accent, for example for hover states. In HSL, hue 203, saturation 100%, lightness 75% yields hex #7DC4FF.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern aesthetic.
- Simple, professional icons to represent technologies and links.
- Responsive layout to ensure compatibility across devices.
- Subtle transitions and animations for a smooth user experience.
